# BilibiliParser —— 解析哔哩哔哩各种信息并打印在MC服务器上
就类似于QQ机器人上的哔哩哔哩视频解析，即发送av/bv号，自动解析信息并打印

## Advantage - 优点
- 发送av号或bv号或链接会自动判断且询问是否要解析
- 好像无了

## depend - 依赖库
- requests
- ruamel

**需要新版本的MCDR(要支持rtext)**

## command - 命令
| 命令 | 功能 |
| ------------ | ------------ |
| !!blbl help | 查看 Bilibili Parser 命令帮助 |
| !!blbl video <av号/bv号/视频链接> | 解析视频信息 |
